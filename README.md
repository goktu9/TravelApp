# Mobile Programming - React Native Lab 3
## Project: TravelApp

This project is a multi-screen "Travel Guide" application built with React Native and Expo as part of the CMP437 Application Development On Android Devices 2025-2026 lab coursework.

The project demonstrates fundamental React Native navigation concepts, including implementing `Stack Navigator` for drilling down into content, `Bottom Tab Navigator` for main menu navigation, passing data between screens using route parameters, and customizing the user interface with `Ionicons` and header styling.

### How to Run

1.  Clone the repository (or navigate to the project root directory).
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the Expo development server:
    ```bash
    npx expo start
    ```
4.  Scan the QR code using the Expo Go app on your iOS or Android device.
